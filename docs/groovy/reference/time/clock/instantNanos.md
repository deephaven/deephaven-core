---
title: instantNanos
---

`instantNanos` returns the current Instant.

## Syntax

```
instantMillis()
```

## Parameters

This method takes no arguments.

## Returns

The Instant.

## Examples

```groovy order=:log
println currentClock().instantMillis()
```
