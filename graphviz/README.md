### Development

```shell
./gradlew graphviz:installDist
```

Run:

```shell
./graphviz/build/install/graphviz/bin/graphviz --help
```

### Build

```shell
./gradlew graphviz:build
```

Artifacts:

* `graphviz/build/distributions/graphviz-<version>.tar`
* `graphviz/build/distributions/graphviz-<version>.zip`
